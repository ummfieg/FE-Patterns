# Frontend Patterns

자주 사용되는 FE Patterns을 모아두었습니다.

---

```
/project-root
├── src/
│   ├── components/
│   │   ├── ui/          # 버튼, 모달 등 재사용 UI (작고 명확한 컴포넌트)
│   │   ├── layout/      # 헤더/푸터, 페이지 레이아웃 (sidebar, MainLayout)
│   │   └── patterns/    # 패턴 학습(네비, 모달, 슬라이더 등)
│   ├── pages/           # 라우팅 기반 페이지
│   ├── hooks/
│   ├── utils/
│   ├── styles/
│   ├── experiments/     # 새로운 기능 실험 폴더(실험 후 components로 배치)
│   ├── App.jsx
│   └── main.jsx
├── README.md
└── package.json
```
